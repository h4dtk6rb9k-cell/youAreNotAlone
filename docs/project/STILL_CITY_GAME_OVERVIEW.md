# Still City - Game Overview

## Concept

Still City is a short atmospheric mobile 2D narrative game about a clean, functioning city where people have not died, rebelled, or been conquered. They have slowly stopped needing to be themselves.

The world did not collapse. Comfort, endless content, passive consumption, and state-supported distraction made personal identity feel unnecessary.

## Theme

The core theme is voluntary fading. The city still works. Windows are warm. Streets are clean. Systems continue. The horror is quiet: personality has become optional, then rare.

A living person should feel precious. If someone truly awake appears, it matters.

## Tone

The tone is restrained, lonely, intimate, and emotionally decayed. The game should not feel like a disaster story. It should feel like waking inside a life that has continued without you.

## Visual Direction

- 2D fake-isometric / 45-degree visual language.
- Clean modern city spaces.
- Warm windows against cold streets.
- Believable human scale.
- Quiet atmosphere, minimal motion, and readable silhouettes.
- Placeholder visuals are allowed during blockout, but geometry must not be absurd.

## Core Gameplay

- Movement.
- Exploration.
- Interaction.
- Minimal dialogue.
- Presence-based NPC reactions.
- Environmental storytelling.
- Level transitions.
- Atmosphere, silence, and rare human contact.

## What The Game Is

Still City is a compact narrative experience designed for mobile portrait play. It uses simple spatial interaction to carry emotional pacing. Each level is a replaceable scene using shared core systems for player movement, interaction, dialogue, transitions, state, and audio mood.

## What The Game Is Not

- Not combat-based.
- Not an RPG.
- Not an inventory game.
- Not a crafting or shop game.
- Not an open world.
- Not HUD-heavy.
- Not a ruins or wasteland story.
- Not a direct copy of any influence.
